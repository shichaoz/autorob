I. Motivation

Autonomous robots, which is mostly about sensing, reasoning and acting, are playing an increasingly essential role in various fields. Sensing allows the robot to construct the maps of its surrounding. After getting the map, the robot can use the method of path planning to generate a path from the starting point to the goal without collision. In order to find the shortest path efficiently, several path planning algorithms are created, such as depth-first search, breadth-first search, A* shortest path algorithm and greedy-best-first search. Without generating the desired path, the robot is not able to take action to move from the starting point to the goal.


II. Method

The four path planing algorithms, including depth-first search, breadth-first search, A* shortest path algorithm and greedy-best-first search are most commonly used. In this report,we apply the four kinds of algorithms to different maps with various obstacles and compare the result with each other.

a) Depth-first search

In the depth-first search algorithm, the element with the highest priority in the visit queue is the one most recently put into the visit queue. Therefore, the search will keep going in one direction, and change direction when hitting the obstacle or collison while obeying the next one in the visit queue.

b) Breadth-first search

In the breadth-first search algorithm, the element with the highest priority in the visit queue is the one most early put into the visit queue. Therefore, the search will keep increasing its search range equally in all direction.

c) A* shortest path algorithm

In the A* algorithm, the element with the highest priority in the visit queue is the one with lowest f-score. F-score of a node equals the sum of the known shortest path from the node to the start and the line distance between the node and the goal. Therefore, the search will spread its search range faster in the direction towards the goal.

d) Greedy-best-first search

In the greedy-best-first search, the element with the highest priority in the visit queue is the one with lowest h-score. H-score of a node equals to the line distance between the node and the goal. Therefore, the search will go directly towards the goal unless hitting the obstacles or collison.

IV. Conclusion

The advantages and the disadvantages of the four algorithms are obvious from the results. Firstly, the Depth-first search can hardly find the shortest path and almost search all the points on the map, which greatly depends on how the priority, often direction, is chosen. The A* algorithm improves the effiency because it gives a relatively reasonable priority to guide the search direction. Both A* and the breadth-first search can find the shortest path. Moreover, Greedy-best-first search is the one with the highest efficiency among A*, breadth-first, and greedy-best-first. However, it very depends on the what the map looks like. If there are obstacles cutting the line between tha start and the goal, greedy-best-first very likely can't find the shortest path. So basically the most practical method for path planning is A* algorithm.
